# 🌌 Nebula Launcher

**Nebula** is a next-generation Minecraft launcher focused on performance, simplicity, and a clean modern experience.

---

## 📸 Preview

A sleek nebula-themed interface with a premium animated design.

---

## 🛠️ Installation

1. Download the `.exe` file
2. Run the installer
3. Launch Nebula
4. Log in and play

---

## ⚠️ Requirements

* Windows 10 / 11
* Java (if not bundled)
* Internet connection
